# Проект Mesto фронтенд + бэкенд

### Описание проекта

Учебный проект для отработки навыков программирования на javascript и чуток html и css.

**Технологии**

В работе применены:

- html
- css
- js
- БЭМ
- ООП
- модульный подход
- асинхронность
- запросы к серверу
- webpack
- React
- роутинг
- авторизация
- Backend на Node.js
- Express.js
- MongoDB, Mongoose, Compass
- Postman
- iron butt
